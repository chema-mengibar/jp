- dandan = poco a poco
- suki ni narimashita = pasó a gustarme / empezó a gustarme
- dekakeru = salir (de casa)
- nandemo ii = cualquier cosa está bien / daba igual
- sugoshimasu = pasar el tiempo
- iru = estar (para seres vivos)
- no ga = convierte el verbo en un sustantivo ("el hecho de estar...")
- kamo shiremasen = quizá / puede que

#  ～たり ～たりする
eiga o mitari = ver películas, por ejemplo
hon o yondari = leer libros, entre otras cosas
tomodachi to paatii shitari = hacer fiestas con amigos

～たり ～たりする se usa para enumerar ejemplos de acciones, con el sentido de "hacer cosas como...". No implica que sean las únicas actividades.
