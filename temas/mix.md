# MIX

## Gramática

* のほうが : comparación, preferencia

  * でんしゃのほうがすきです

* ために : para, con el fin de

  * やさいなどをそだてるために

* ～というものです : consiste en..., se trata de...

  * それぞれのすぺーすをかすというものです

* ～という : llamado...

  * せびっちぇというりょうり
  * しゅれーばーがるてんというにわ

* ～といいます : se llama...

  * これはそぶれめさといいます

* ～そうです : parece...

  * そんなぱーてぃたのしそうです

* ～ので : porque...

  * ひつじややぎがいるので
  * たくさんあるので

* ～かもしれません : quizás...

  * もういかないかもしれません

* ～のように : como...

  * はなのようにひらきます

* もともと : originalmente

  * もともとたべていなかったです
