# ため (tame)

No lo confundas con kara.

- kara = porque (causa)
- tame ni = para (objetivo)

Por ejemplo:

Atsui desu kara mizu o nomimasu. -> Bebo agua porque hace calor.
Karada no tame ni mizu o nomimasu. -> Bebo agua para cuidar mi cuerpo.

## 1. Para... / con el fin de...

Se usa para expresar un objetivo.

Ejemplos:

Nihongo o benkyou shimasu. Nihon e iku tame desu.
= Estudio japonés para ir a Japón.

〜のために
Karada no tame ni undou shimasu.
= Hago deporte por la salud (literalmente: para el cuerpo).


## 2. Con verbos

Se usa la forma diccionario.

Por ejemplo:

iku tame ni = para ir
hataraku tame ni = para trabajar
kau tame ni = para comprar

Nihon e iku tame ni, okane o tamemasu.

= Ahorro dinero para ir a Japón.
